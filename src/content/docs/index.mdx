## Learning taxonomy registry

<CardGrid stagger>
	<Card title="Stable IDs" icon="link">
		The taxonomy registry publishes canonical URIs for taxonomy schemes, levels, and verbs under `/taxonomy/...`.
	</Card>
	<Card title="Content negotiation" icon="server">
		Taxonomy identifiers return HTML, JSON, JSON-LD, or Turtle based on the
		HTTP `Accept` header.
	</Card>
	<Card title="Registry-driven" icon="open-book">
		YAML files in `/registry` compile into the taxonomy knowledge graph at build
		time.
	</Card>
	<Card title="SKOS output" icon="document">
		Taxonomies publish SKOS concept schemes and concepts with Instruction as Code
		ontology extensions.
	</Card>
	<Card title="Crosswalks" icon="seti:graph">
		Cross-taxonomy mappings use SKOS mapping predicates such as `skos:closeMatch`.
	</Card>
</CardGrid>
